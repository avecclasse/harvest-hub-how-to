# Harvest Hub — Build Tutorial

This is your personalized tutorial for building **Harvest Hub** (a produce reservation app with a community capacity pool and order tracking) using Next.js, InstantDB, Tailwind CSS, and Vercel.

## Viewing This Tutorial

### Quick Start: View Locally

1. Open `index.html` in your web browser (double-click the file or drag it into the browser window).
2. Use the top navigation to move between pages: **Introduction**, **Setup**, **Build**, **Deploy**, and **Reference**.

### Host on GitHub Pages (Recommended)

Hosting the tutorial on GitHub Pages gives you a permanent URL you can share or bookmark.

#### Step 1: Create a GitHub Repository

1. Go to [github.com](https://github.com) and sign in.
2. Click the **+** in the top right → **New repository**.
3. Name it (e.g. `harvest-hub-tutorial`).
4. Select **Public**.
5. Click **Create repository**.

#### Step 2: Upload Files

1. On your new repository page, click **“uploading an existing file”** (or “Add file” → “Upload files”).
2. Drag all files from this folder into the browser:
   - `index.html`
   - `setup.html`
   - `build.html`
   - `deploy.html`
   - `reference.html`
   - `styles.css`
   - `scripts.js`
   - `README.md`
3. Click **Commit changes**.

#### Step 3: Enable GitHub Pages

1. In the repository, go to **Settings** → **Pages**.
2. Under **Source**, select **Deploy from a branch**.
3. Under **Branch**, choose **main** (or **master**) and **/ (root)**.
4. Click **Save**.

#### Step 4: View Your Tutorial

After a minute or two, your tutorial will be live at:

```
https://YOUR-USERNAME.github.io/REPOSITORY-NAME/
```

Replace `YOUR-USERNAME` with your GitHub username and `REPOSITORY-NAME` with the name you gave the repo (e.g. `harvest-hub-tutorial`).

---

## Next Steps

1. Work through the tutorial in order: **Introduction** → **Setup** → **Build** → **Deploy** → **Reference**.
2. Copy the **Build Prompt** from the **Build** page into Cursor’s Composer (and replace `YOUR_APP_ID_HERE` with your InstantDB App ID).
3. Deploy your finished Harvest Hub app to Vercel using the **Deploy** page.
4. Share your app and this tutorial with others when you’re ready.

Good luck!
